Run Simulation:
1. Execute the simulation program through single-step mode until a mesh computer is created.
2. Select an event named "EC_START", right click it, and choose <Fast run until message "EC_START"> or <Express run until message "EC_START">.
3. Open all clients, and change the flag of "connection" from value <false> to <true>.
4. Click the "EXPRESS" button to run the simulation until the end of it
